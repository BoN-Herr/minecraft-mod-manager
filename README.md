# FriendsPack — Client Setup

1. Get `mcclient.exe` and the server address (e.g. `192.168.1.50:8080`) from your host.
2. Double-click `mcclient.exe` (if Windows warns, click **More info → Run anyway**).
3. Type the server address when prompted and press **Enter**.
4. Wait while it installs Forge and downloads the mods.
5. Open the Minecraft Launcher, pick the pack installation, and click **Play**.
6. To update later, just run `mcclient.exe <address>` again.
